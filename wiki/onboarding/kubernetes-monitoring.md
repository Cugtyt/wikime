# Kubernetes Monitoring Stack

Overview of the monitoring tools used for Kubernetes cluster observability, as covered during team onboarding.

## Grafana Dashboards

Grafana is the primary visualization layer for cluster metrics. New team members (e.g., Jake) are onboarded by walking through the existing dashboards that cover cluster health, pod status, and resource usage.

## Key Takeaways

- Grafana dashboards are the go-to tool for Kubernetes observability
- Onboarding new team members should include a walkthrough of the monitoring stack
- Monitoring knowledge helps with debugging issues like CNI problems
